# **App Name**: XLSX Viewer

## Core Features:

- File Upload: Allow users to upload .xlsx files.
- Data Extraction: Parse the .xlsx file and extract data from the first column.
- List View Generation: Dynamically generate a list view to display the extracted data.
- Error Handling: Display user-friendly error messages for invalid file formats or corrupted data.

## Style Guidelines:

- Background color: Dark gray (#121212) for a modern dark theme.
- Primary color: Soft blue (#64B5F6) to ensure good contrast on the dark background, offering visual clarity and ease of reading.
- Accent color: A more vibrant cyan (#00BCD4) is selected to highlight interactive elements.
- Font: 'Inter', a sans-serif font that ensures excellent readability and a clean aesthetic, perfect for displaying data.
- Use a clean and simple layout with a prominent file upload button and a clearly defined list view area.
